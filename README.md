## Button
| 코드                     | 기능           |
|-------------------------|-----------------|
| .padding() | 여백주기, 괄호안에 숫자 넣어도 된다. |
| .background() | 백그라운드 색 주기 |
| .cornerRadius() | 윤곽선 둥글게 만들기 |
| .frame() | 크기 바꾸기 |
| .foregroundColor() | 텍스트나 이미지 또는 버튼 색 변경, Button의 컨테이너 전체에 적용되진 않음 |
